Swapping

Existe un algoritmo llamado algoritmo de intercambio. Este es el que decide si una página se desecha o se envía al disco.


Reemplazo

Linux utiliza la técnica de reemplazo de marco de página por antigüedad (Last Recently Used, LRU) para escoger de forma equitativa y justa las páginas a ser intercambiadas o descartadas del sistema.


Gestión del dispositivo SWAP

Un dispositivo de swap debe haber sido inicializado por la orden mkswap. Esta orden crea un directorio al arrancar el dispositivo.

SWAPON #include #include int swapon(const char *pathname, int swapflags); SWAPOFF #include int swapoff(const char *pathname);


Entrada/Salida de páginas de Swap

Existe una función llamada rw_swap_page, está lee o escribe las páginas en un dispositivo swap. Este verifica que el dispositivo está activo, que el número de página es válido.


Eliminacion de paginas de memoria swap

Existen varias maneras para eliminar una pagina de un dispositivo Swap · try_to_swap_out: esta se llama para intentar eliminar una página específica · page_unuse: cuando la página no ha sido modificada y se suprime de la chaché de paginas · swap_out_process: se llama para eliminar una página del espacio de direccionamiento de un proceso.
